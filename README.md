# 🖼️ Image Gallery

A simple image gallery built with React as my first frontend project. This project helped me understand the fundamentals of React by creating reusable components and rendering data dynamically.

## 🚀 About the Project

This is a single-page image gallery application that displays a collection of images stored in the project's assets folder. The main goal of this project was to practice React basics and understand how component-based development works.

## 📚 What I Learned

Through this project, I learned:

- Setting up a React project
- Creating and using React components
- Passing data using **Props**
- Rendering lists with the **map()** method
- Organizing project files and folders
- Importing and displaying local images
- Understanding JSX syntax

## 🛠️ Technologies Used

- React
- JavaScript (ES6+)
- HTML
- CSS

## 📸 Features

- Simple and clean UI
- Displays images from the assets folder
- Uses reusable React components
- Dynamically renders images using `map()`
- Passes image data through props
